# Spelman — Roadmap

Where we're headed. Updated as priorities shift.

---

## Next Up

- **Crossfade** — smooth transitions between tracks
- **Sorting & search** — sortable queues/playlists, fuzzy search across any list
- **Metadata browser** — column-based library (Artist → Album → Track)
- **MPRIS media keys** — system play/pause/next on Linux

## Planned

- **Lyrics display** — synced lyrics from tags or .lrc files
- **Metadata editor** — fix tags in-app, AI-assisted bulk correction
- **Audio device negotiation** — automatic sample rate matching with software resampling fallback
- **Broader codec support** — ALAC, AIFF, Opus, MKV, and more via Symphonia feature flags

## Exploring

- **Streaming integrations** — Archive.org, Radio-Browser, Audius, Jamendo, and others via a unified streaming interface
- **AI features** — local Ollama for mood-based playlists, auto-DJ, natural language search
- **AI music generation** — Google Lyria, ACE-Step, BYOK services
- **Vindharpa** — GUI frontend with Spelman as a background service (IPC/socket control)
- **Desktop notifications** — notify-send / osascript on track change
